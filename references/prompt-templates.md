# Image Generation Prompt Templates

These are concept-anchored prompt templates — not fill-in-the-blank forms. Every image prompt must be built from the Creative Brief's campaign concept, not just brand colors and dimensions.

The difference between a forgettable post and a scroll-stopping one is whether the image expresses an IDEA or just looks "on brand."

---

## The Three-Layer Prompt System

Every image prompt must contain these three layers, in order. The first two layers are the most important — they're what most AI-generated social content gets wrong.

### Layer 1: Campaign Concept Anchor

Start by stating how THIS image expresses the campaign's core idea. This ensures the image serves the campaign, not just the brand.

```
CONCEPT: This image visualizes [specific aspect of campaign concept].
The viewer should understand [what revelation/idea] at first glance.
```

**Example for a campaign concept "In a world where seeing isn't believing, proof is power":**
- Product showcase: "This image visualizes the moment of verification — the instant uncertainty becomes certainty."
- Data stat: "This image visualizes the scale of the problem — how much of what we see online can't be trusted."
- Testimonial: "This image visualizes the relief of having proof — the comfort of knowing, not guessing."

If you can't articulate how the image connects to the campaign concept, the post doesn't belong in this campaign.

### Layer 2: Emotional Intent

Specify what the viewer should FEEL. Emotional intent shapes every design decision — composition, color temperature, typography weight, negative space.

```
EMOTION: The viewer should feel [specific emotion] — [one sentence describing the moment].
```

**Good emotional intents:**
- "The viewer should feel a jolt of unease — the realization that they can't tell real from fake"
- "The viewer should feel quiet confidence — the reassurance of having a tool that sees through deception"
- "The viewer should feel urgency — the sense that this problem is getting worse every day"
- "The viewer should feel curiosity — an itch to swipe and learn more"

**Bad emotional intents (too vague):**
- "Professional and modern" (that's a style, not an emotion)
- "Engaging" (meaningless)
- "Eye-catching" (describes the goal, not the feeling)

### Layer 3: Visual Execution

Now specify the concrete visual details. These should flow naturally from the concept and emotion above.

```
SCENE: [Concrete visual description anchored to the concept]
STYLE: [Brand style], [aesthetic keywords from Visual Identity System]
COLORS: Primary [#hex], Accent [#hex], Background [#hex] — used as [how, from Visual Identity System]
TYPOGRAPHY: [Text overlays — must echo the caption's hook thematically]
DIMENSIONS: [From platform-specs.md]
ANTI-PROMPT: Do NOT include [specific things to avoid — generic stock imagery, etc.]
```

## Template Library

Use these as starting frameworks, but always build from the three layers above. The concept and emotion layers come first — these templates handle Layer 3.

### 1. Product Showcase

**When to use:** Demonstrating what the product does or how it works.

**Why this format works:** People process images 60,000x faster than text. A product showcase should create a "moment of use" — the viewer should imagine themselves using it.

**Concept connection step:** Before writing the prompt, answer: "What does using this product FEEL like at the exact moment it delivers value?"

```
CONCEPT: [How this showcase connects to the campaign's core idea]
EMOTION: [What the viewer should feel — usually empowerment, relief, or "I need this"]

A premium product showcase for [Platform], [dimensions].
[Brand visual style] aesthetic with [primary color] (#hex) as the dominant tone.
Feature [product name] in the moment of delivering value — not just a static UI screenshot,
but the MOMENT where [the key transformation happens].
[Describe the specific "moment of value" — e.g., "the scan result revealing the truth about an image"]
Subtle gradient from [primary color] to [lighter variant] creating depth.
Text overlay: "[Line that echoes the caption's hook]" in [accent color] (#hex).
Brand logo subtle, bottom-right, 20% opacity.
The image should feel like a snapshot of the product doing its most impressive thing.
Do NOT include: static UI screenshots with no context, generic dashboard views, busy backgrounds.
```

**Bad prompt example:**
"A product showcase of Proofly with blue background and white text showing the logo."

**Good prompt example:**
"CONCEPT: This image shows the moment of truth — the instant where the user sees proof of what's real.
EMOTION: The viewer should feel a flash of revelation — the 'aha' moment of certainty.
A premium product showcase for Instagram, 1080×1350px. Dark, sophisticated aesthetic with deep navy (#0a1628) background. The hero element is a verification result card emerging from shadow into light — showing an image being analyzed with a clear 'AI Generated — 94% confidence' result. The card has a subtle green glow at the edges. The surrounding darkness represents uncertainty; the illuminated card represents truth. Text overlay: '3 seconds to truth.' in crisp white, top-third. Logo watermark bottom-right at 15% opacity. This should feel like a flashlight cutting through darkness."

### 2. Feature Highlight / Education

**When to use:** Teaching the audience something or spotlighting a specific feature.

**Why this format works:** Educational content earns saves and shares — the highest-value engagement signals.

**Concept connection step:** Before writing the prompt, answer: "What does the audience NOT know that would change how they think about this problem?"

```
CONCEPT: [How this educational moment connects to the campaign's core idea]
EMOTION: [Usually curiosity, surprise, or "I didn't know that"]

A clean, informative social graphic for [Platform], [dimensions].
[Brand style] design on [background color] (#hex).
[Describe a visual metaphor or comparison that teaches the concept — NOT just text on a background]
The educational point should be VISIBLE in the image, not just stated in text.
Heading: "[Feature/Insight Name]" in [brand heading font], [accent color] (#hex).
2-3 visual supporting elements (icons, mini-illustrations, comparison elements).
The overall feeling should be "I learned something useful" not "I'm being marketed to."
Do NOT include: walls of text, complex diagrams, more than 25 words total on the image.
```

### 3. Testimonial / Social Proof

**When to use:** Building trust through other people's words and experiences.

**Why this format works:** People trust other people more than brands. But generic quote cards feel manufactured.

**Concept connection step:** Before writing the prompt, answer: "How does this person's experience prove the campaign concept is true?"

```
CONCEPT: [How this testimonial reinforces the campaign's core idea]
EMOTION: [Usually trust, belonging, or "if it works for them, it'll work for me"]

An elegant testimonial card for [Platform], [dimensions].
[Background color] (#hex) with a softly rounded card in [surface color] (#hex).
The design should feel genuine and warm, not corporate.
Large quotation mark as decorative element in [accent color] (#hex), 40% opacity.
Quote: "[Customer quote that specifically proves the campaign concept]" in [text color], medium weight.
Customer name and context below — make the person feel REAL (name + role, not just initials).
Optional: a subtle visual element that connects to the product (e.g., verification check, shield icon).
Minimal and spacious — the words are the hero.
Do NOT include: fake-looking stock elements, cluttered design, generic 5-star graphics.
```

### 4. Announcement / Launch

**When to use:** Big news, new features, milestones, launches.

**Why this format works:** Announcements create FOMO and community — "this is happening, and you want to be part of it."

**Concept connection step:** Before writing the prompt, answer: "Why should someone feel excited about this news — not because it's new, but because of what it means?"

```
CONCEPT: [How this announcement advances the campaign's story]
EMOTION: [Usually excitement, anticipation, or "this changes things"]

A bold announcement graphic for [Platform], [dimensions].
Dynamic composition with [primary color] (#hex) commanding attention.
Large typography: "[Announcement — as a statement, not just info]" in white or [contrast color].
Supporting detail in smaller type — only the essential info (date, one action).
Energy elements: [choose one — radial burst, confetti particles, glow effect] in [accent color] (#hex).
CTA element: rounded pill in [accent color] with "[specific action]" in white.
The image should make someone feel they're seeing something IMPORTANT happen.
Do NOT include: multiple CTAs, dense information, passive language, "we're excited to announce".
```

### 5. Behind-the-Scenes / Brand Story

**When to use:** Humanizing the brand, showing the work behind the product.

**Why this format works:** Vulnerability and authenticity build deeper connection than polished marketing ever can.

**Concept connection step:** Before writing the prompt, answer: "What part of our process/story would make someone feel connected to the humans behind this product?"

```
CONCEPT: [How this behind-the-scenes moment connects to the brand's mission]
EMOTION: [Usually warmth, relatability, or admiration for craftsmanship]

A warm, authentic behind-the-scenes image for [Platform], [dimensions].
Editorial photography aesthetic — slightly desaturated, natural tones.
Scene: [Describe a SPECIFIC moment that reveals craft or mission — not "team working at desks"]
Natural lighting, no obvious staging.
Brand color accent appearing organically in the scene — a [primary color] element.
No text overlay — the image speaks, the caption tells the story.
The feeling should be "I just got a look behind the curtain and I like what I see."
Do NOT include: corporate stock photo aesthetic, forced smiles, overly staged scenes.
```

### 6. Carousel / Multi-Slide Post

For carousel posts, visual consistency across slides is critical. All slides must feel like pages of the same book.

**Concept connection step:** Before writing, answer: "What's the ONE argument this carousel makes, and what's each slide's role in making it?"

#### Cover Slide (Hook)

```
CONCEPT: [The question or tension the carousel will explore]
EMOTION: [Curiosity — must compel a swipe]

A bold cover slide for [Platform] carousel, [dimensions].
[Primary color] (#hex) background with large, provocative text:
"[Hook — a question, a bold claim, or a number that demands explanation]"
Swipe indicator: subtle arrow or "Swipe →" in [accent color] at bottom.
This slide has ONE job: make them swipe. Nothing else.
Brand watermark in corner at 20% opacity.
Do NOT include: too much information, answers to the hook, more than 10 words.
```

#### Content Slides (2-N)

```
Consistent with the cover: same margins, fonts, spacing, color system.
Each slide makes ONE point — no cramming.
Point heading in [accent color], bold, top-third.
Supporting text or visual below, concise.
Slide N of [total] indicator in bottom corner.
Visual continuity: same background, same text sizing, same layout grid.
```

#### Final CTA Slide

```
EMOTION: [Motivation — the viewer should feel compelled to act]

Visual shift: [accent color] (#hex) background for contrast — signals the end.
Text: "[CTA — specific, action-oriented, not 'follow for more']" in white.
Subtext: "[Where to go — 'Link in bio', 'DM us', 'Try free']" in white, smaller.
Brand logo centered, full opacity.
This slide should feel conclusive — "now it's your turn."
```

### 7. Comparison / Before-After

**Concept connection step:** Before writing, answer: "What's the most dramatic contrast between 'without us' and 'with us'?"

```
CONCEPT: [What transformation this comparison proves]
EMOTION: [Usually a combo — discomfort on the left, relief on the right]

Split layout for [Platform], [dimensions].
Left: "[Without/Before]" — muted, gray-toned, visually uneasy.
[Describe the specific "before" scene — make the problem VISIBLE]
Right: "[With/After]" — vibrant, [brand colors], polished, resolved.
[Describe the specific "after" scene — make the solution VISIBLE]
Thin divider in [accent color] (#hex).
The difference should be obvious in ONE second — no reading required.
Do NOT include: text-heavy comparisons, subtle differences, cluttered layout.
```

### 8. Statistics / Data Point

**Concept connection step:** Before writing, answer: "Why should this number make someone FEEL something? What does it mean for THEM?"

```
CONCEPT: [What truth this data point proves about the campaign's thesis]
EMOTION: [Usually shock, urgency, or vindication]

A striking data visualization for [Platform], [dimensions].
[Background color] (#hex) with the statistic as the hero:
"[Number]" in oversized [accent color] (#hex), centered, commanding.
Context line below: "[What this means in human terms]" in [text color].
Subtle data-viz element in background (abstract lines, dots) at 8% opacity.
Source citation, small, bottom — this isn't made up.
The number should hit like a headline.
Do NOT include: complex charts, multiple statistics, unattributed data.
```

## Prompt Quality Checklist

Before finalizing any image prompt, verify:

- [ ] Layer 1 present: Concept anchor connects image to campaign idea
- [ ] Layer 2 present: Emotional intent is specific (not "professional" or "engaging")
- [ ] Layer 3 present: Visual details flow from concept and emotion
- [ ] Text overlay echoes caption hook (they should feel inseparable)
- [ ] Visual Identity System references are consistent (motif, color rules, typography)
- [ ] Anti-prompt included (what to specifically avoid)
- [ ] Competitor swap test passed (this image couldn't belong to a competitor)

## Prompt Modifiers

Add these to any template to adjust the output:

| Modifier | Effect |
|----------|--------|
| `"Flat design, no shadows or gradients"` | Minimalist, modern look |
| `"Glassmorphism effect on card elements"` | Trendy, tech-forward |
| `"Grain texture overlay at 5% opacity"` | Vintage, editorial feel |
| `"Neon glow on accent elements"` | Bold, nightlife, gaming aesthetic |
| `"Watercolor texture in background"` | Soft, artistic, lifestyle brands |
| `"Duotone color treatment"` | Strong brand recognition |
| `"Isometric perspective"` | Tech/SaaS, 3D feel without 3D |
| `"Hand-drawn illustration style"` | Friendly, approachable, playful |
