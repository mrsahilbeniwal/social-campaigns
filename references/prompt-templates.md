# Image Generation Prompt Templates

Ready-to-adapt prompt templates for common social media post types. Replace bracketed values with brand-specific details from the Brand Profile.

---

## 1. Product Showcase

```
A premium product showcase image for [Platform], [dimensions].
[Brand visual style] aesthetic with [primary color] (#hex) background.
Feature [product name/type] as the hero element, centered with generous breathing room.
Subtle gradient from [primary color] to [slightly lighter/darker variant].
Clean sans-serif text overlay: "[Product tagline or key benefit]" in [accent color] (#hex).
Brand logo small in the bottom-right corner.
[Brand mood] feeling — professional yet approachable.
Soft directional lighting from top-left creating gentle shadows.
Do NOT include: busy backgrounds, multiple products, stock-photo styling.
```

## 2. Feature Highlight / Tip

```
A clean, informative social media graphic for [Platform], [dimensions].
[Brand style] design on [background color] (#hex) background.
Left side: icon or minimal illustration representing [feature/tip topic] in [accent color] (#hex).
Right side: Bold heading "[Feature Name]" and 2-3 bullet points of key benefits.
Typography: [Brand heading font] for title, [Brand body font] for bullets.
Subtle decorative element — thin line or dot pattern in [secondary color] (#hex) at 10% opacity.
Educational, clear, easy-to-scan at a glance.
Do NOT include: complex diagrams, photographs, more than 30 words total.
```

## 3. Testimonial / Social Proof

```
An elegant testimonial card for [Platform], [dimensions].
[Background color] (#hex) base with a softly rounded card in [surface color] (#hex).
Large opening quotation mark in [accent color] (#hex) as a decorative element.
Quote text: "[Customer quote]" in [text color] (#hex), italic weight.
Below the quote: Customer name and title/company in smaller text.
Optional: 5 star icons in [accent color] (#hex) above the quote.
Warm, trustworthy, authentic feel matching [brand personality].
Minimal and spacious — the words are the hero.
Do NOT include: customer photos (privacy), fake-looking elements, cluttered design.
```

## 4. Announcement / Launch

```
A bold, attention-grabbing announcement graphic for [Platform], [dimensions].
Dynamic composition with [primary color] (#hex) as dominant background.
Large, impactful typography — "[Announcement headline]" in white or [contrast color].
Supporting subtext in smaller size: "[Details — date, link, key info]".
Energy-creating elements: subtle radial burst, confetti particles, or glow effect in [accent color] (#hex).
CTA button graphic element: rounded rectangle in [accent color] with "[CTA text]" in white.
Exciting, celebratory, impossible to scroll past.
Do NOT include: too many details (save for caption), dull colors, passive language.
```

## 5. Behind-the-Scenes / Brand Story

```
A warm, authentic behind-the-scenes style image for [Platform], [dimensions].
Slightly desaturated, editorial photography aesthetic.
Scene showing [workspace/process/team element relevant to brand].
Natural, soft lighting — as if shot during golden hour.
Muted version of brand colors in the environment — [primary color] (#hex) accent on a desk item or screen.
No text overlay — let the image speak (caption carries the story).
Genuine, relatable, human feel that builds brand connection.
Do NOT include: overly staged elements, corporate stock-photo feel, harsh lighting.
```

## 6. Carousel / Multi-Slide Post

For carousel posts, generate each slide individually with consistent styling:

### Slide 1 (Cover/Hook):
```
A bold cover slide for a [Platform] carousel, [dimensions].
[Primary color] (#hex) background with large white text: "[Hook question or statement]".
Subtle swipe indicator (arrow or "Swipe →") in [accent color] (#hex) at bottom.
Clean, minimal — designed to stop the scroll and invite a swipe.
Brand logo watermark in corner at 30% opacity.
```

### Slides 2-N (Content):
```
Content slide [N] for a [Platform] carousel, [dimensions].
[Background color] (#hex) background, consistent with the cover slide styling.
Section heading: "[Point N title]" in [accent color] (#hex), bold.
Supporting text or bullet points in [text color] (#hex), concise.
Optional: small icon or illustration reinforcing the point.
Visual consistency with previous slides — same margins, font sizes, spacing.
```

### Final Slide (CTA):
```
Final CTA slide for a [Platform] carousel, [dimensions].
[Accent color] (#hex) background for visual contrast — signaling the end.
Text: "[CTA headline — e.g., 'Ready to start?']" in white, large and bold.
Subtext: "[Supporting action — e.g., 'Link in bio' or 'DM us']" in white, smaller.
Brand logo centered or bottom-center at full opacity.
Energetic, conclusive, action-oriented.
```

## 7. Comparison / Before-After

```
A side-by-side comparison graphic for [Platform], [dimensions].
Split layout — left side labeled "Before" / "Without [Product]" and right side "After" / "With [Product]".
Left side: muted, gray-toned, visually "less appealing" representation.
Right side: vibrant, [brand colors], polished representation.
Thin divider line in [accent color] (#hex) between the two halves.
Small labels at top of each half in clean sans-serif.
Clear, immediate visual impact — the difference should be obvious in 1 second.
Do NOT include: text-heavy comparisons, more than 2 comparison points, cluttered layouts.
```

## 8. Statistics / Data Point

```
A striking data visualization graphic for [Platform], [dimensions].
[Background color] (#hex) with a large, bold statistic as the hero element.
Number "[XX%]" or "[X,XXX+]" in oversized [accent color] (#hex) font, centered.
Supporting context line below: "[What this number means]" in [text color] (#hex).
Subtle data-viz element in background — abstract chart lines, dots, or graph shapes at 10% opacity in [secondary color].
Source citation in small text at bottom.
Authoritative, credible, shareable.
Do NOT include: complex charts, multiple statistics, unverified numbers.
```

## Prompt Modifiers

Add these to any template to adjust the output:

| Modifier | Effect |
|----------|--------|
| `"Flat design, no shadows or gradients"` | Minimalist, modern look |
| `"Glassmorphism effect on card elements"` | Trendy, tech-forward |
| `"Grain texture overlay at 5% opacity"` | Vintage, editorial feel |
| `"Neon glow on accent elements"` | Bold, nightlife, gaming aesthetic |
| `"Watercolor texture in background"` | Soft, artistic, lifestyle brands |
| `"Duotone color treatment"` | Strong brand recognition |
| `"Isometric perspective"` | Tech/SaaS, 3D feel without 3D |
| `"Hand-drawn illustration style"` | Friendly, approachable, playful |
