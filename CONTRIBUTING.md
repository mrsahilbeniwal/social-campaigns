# Contributing to Social Campaign

First off — **thank you** for considering contributing to Social Campaign! This project exists because of people like you who believe that marketing shouldn't be gatekept behind expensive agencies and marketing degrees.

## 🌟 How You Can Help

There are many ways to contribute, and not all of them involve writing code:

### 💡 Share Ideas
- Open a [Feature Request](https://github.com/mrsahilbeniwal/social-campaigns/issues/new?template=feature_request.md) with your idea
- Suggest new prompt templates, caption frameworks, or platform specs
- Share what worked (or didn't work) when you used the skill

### 🐛 Report Bugs
- If a prompt template produces poor results, let us know
- If the skill misses important brand elements during scanning, report it
- If platform specs are outdated (social media platforms change constantly!), flag it

### 📝 Improve Content
- Add prompt templates for new post types
- Add content pillar templates for new industries
- Add caption frameworks you've tested and proven
- Update platform specs with the latest dimensions and limits
- Improve the SKILL.md instructions for clarity

### 🌐 Expand Coverage
- Add support for new platforms (Threads, Bluesky, Mastodon, WeChat, etc.)
- Add language support / localization guides
- Add region-specific marketing playbooks

### 📊 Share Results
- If you've used Social Campaign and got real results, share your experience
- Before/after engagement metrics help us understand what works
- Screenshots of generated campaigns (anonymized if needed) help others

## 🔧 How to Contribute

### 1. Fork & Clone

```bash
git clone https://github.com/YOUR-USERNAME/social-campaign.git
cd social-campaign
```

### 2. Create a Branch

```bash
git checkout -b feature/your-feature-name
```

### 3. Make Your Changes

Follow these guidelines:
- **SKILL.md:** Keep it under 500 lines. If adding features, consider putting details in a reference file
- **Reference files:** Each file should cover one topic thoroughly
- **Markdown formatting:** Match the existing style (tables, code blocks, examples)
- **Be specific:** Avoid vague advice. Include exact dimensions, character counts, examples

### 4. Test Your Changes

If you're updating prompt templates or caption frameworks:
- Try them with an actual AI agent (Claude Code, Cursor, etc.)
- Verify the generated content is high quality
- Test with at least 2 different project types (e.g., SaaS + e-commerce)

### 5. Submit a Pull Request

- Fill out the PR template
- Explain what you changed and why
- Include examples of improved output if applicable
- Reference any related issues

## 📋 Style Guide

### Writing Style
- **Be conversational** — Write like you're explaining to a smart friend, not writing a textbook
- **Be specific** — "Use 1080x1350px for Instagram portrait posts" beats "Use the right dimensions"
- **Be practical** — Every addition should be actionable, not theoretical
- **Show examples** — Fill-in-the-blank templates > abstract advice

### File Organization
```
social-campaign/
├── SKILL.md                  # Main skill instructions (keep < 500 lines)
├── README.md                 # GitHub readme
├── LICENSE                   # MIT License
├── CONTRIBUTING.md           # This file
├── references/               # Detailed reference material
│   ├── platform-specs.md     # Platform dimensions & limits
│   ├── prompt-templates.md   # Image generation prompts
│   ├── content-pillars.md    # Industry-specific pillars
│   ├── campaign-types.md     # Campaign playbooks
│   └── caption-frameworks.md # Caption writing formulas
└── examples/                 # Real-world examples (coming soon)
```

### Naming Conventions
- Files: `kebab-case.md`
- Headers: Title Case for H2, Sentence case for H3+
- Templates: Use `[brackets]` for fill-in-the-blank sections

## 🎯 Priority Areas

These are the areas where contributions would have the most impact:

1. **More prompt templates** — We have 8, but there are dozens of post types we haven't covered (polls, event invites, job postings, milestone celebrations, etc.)
2. **Industry-specific pillar packs** — Healthcare, EdTech, FinTech, Food & Beverage, Real Estate, etc.
3. **Regional marketing guides** — What works in the US doesn't always work in India, LATAM, or EMEA
4. **Platform updates** — Social platforms change their specs and algorithms regularly. Help us stay current.
5. **Real-world case studies** — Nothing beats "I used this and here's what happened"

## 🤝 Code of Conduct

Be kind. Be helpful. Be constructive. We're all here to make marketing more accessible.

- Treat everyone with respect
- Provide constructive feedback
- Welcome newcomers — we all started somewhere
- Focus on what's best for the community

## 🏆 Recognition

All contributors will be recognized in the README. Significant contributions get a shoutout in the changelog.
